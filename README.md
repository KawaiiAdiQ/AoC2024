### Description

for each new day:
```bash
# 01 -> 25
cp -r template 01
```

to compile & run individual code:
```bash
# compiles a.cpp
# and runs 'a' < in/a.txt 
make a

# compiles b.cpp
# and runs 'b' < in/b.txt 
make b
```
